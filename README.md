* Project Overview

The Electricity Bill Generator is a Java-based desktop application developed using Swing (GUI) and MySQL database connectivity (JDBC). The application allows users to enter customer details and electricity meter readings, automatically calculates the electricity bill using slab-based pricing, and stores the records in a database.

This project follows Object-Oriented Programming (OOP) principles and a modular architecture for better readability and maintenance.

* Features
User-friendly graphical interface using Java Swing
Input validation for all fields (empty, numeric, logical checks)
Automatic electricity bill calculation using slab system
MySQL database integration using JDBC
Stores customer billing records in database
Clean modular structure using DAO pattern
Error handling for invalid inputs

* Technologies Used
Java (Core + Swing)
JDBC (Java Database Connectivity)
MySQL Database
Object-Oriented Programming (OOP)
